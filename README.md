# 🍽️ FlavorNest – Global Kitchen

FlavorNest – Global Kitchen is a modern, fully responsive recipe discovery website that allows users to explore dishes from around the world.

It is built using simple web technologies and focuses on clean design, smooth user experience, and useful features like search, filtering, and favorites.

---

## 🚀 Features

- Explore recipes from multiple cuisines
- Search recipes by name or ingredients
- Filter recipes by culture (Indian, Italian, Chinese, etc.)
- Save favorite recipes using browser storage
- Fully responsive (mobile, tablet, desktop)
- Modern UI with smooth animations

---

## 🧑‍🍳 Supported Cuisines

- Indian
- Italian
- Chinese
- Mexican
- Japanese
- French
